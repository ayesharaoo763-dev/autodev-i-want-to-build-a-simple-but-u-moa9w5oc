# To-Do List App
A simple and clean To-Do List app for managing daily tasks.

## Features
* Add, edit, and delete tasks
* Mark tasks as completed
* Set due dates for tasks
* Task prioritization (low, medium, high)
* Simple and clean UI

## Usage
1. Open the app in a web browser.
2. Add new tasks using the form at the top of the page.
3. Mark tasks as completed by checking the box next to the task.
4. Edit tasks by clicking the edit button next to the task.
5. Delete tasks by clicking the delete button next to the task.

## Deployment
The app is deployed on Vercel and can be accessed at [https://todo-list-app.vercel.app](https://todo-list-app.vercel.app).
